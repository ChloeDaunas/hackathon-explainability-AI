## рџ“Љ Data

**Source**: [Human Resources Dataset вЂ” Kaggle (Rich Huebner & Carla Patalano)](https://www.kaggle.com/datasets/rhuebner/human-resources-data-set)

- ~311 employees (after anonymization)
- Target variable: `Termd` (0 = active, 1 = left)
- Turnover rate: ~33%
- Sensitive attributes present: `Sex`, `RaceDesc`, `GenderID`

**Data Card вЂ” Transformations applied:**

| Technique | Columns affected | Justification |
|-----------|-----------------|---------------|
| Suppression | `Employee_Name`, `ManagerName` | Direct identifiers |
| Pseudonymization | `EmpID` | Truncated SHA-256 hash, non-reversible |
| Generalization | `DOB` в†’ `AgeGroup` | Useful info without exact date |
| Masking | `Zip` в†’ first 2 digits | Reduction of geographic precision |
| Model exclusion | `Sex`, `RaceDesc`, `GenderID` | Retained only for fairness audit |
